Gerenciador de banco de dados MySQL

Gerenciamento da Amazon

O MySQL é o banco de dados relacional de código aberto mais popular do mundo e o Amazon RDS facilita a configuração, a operação e a escalabilidade de implantações de MySQL na nuvem. Com o Amazon RDS, você pode implantar em minutos servidores MySQL escaláveis com capacidade de hardware econômica e redimensionável. O Amazon RDS for MySQL gerencia as demoradas tarefas de administração de banc o de dados, incluindo backups, aplicação de patches, monitoramento, escalabilidade e replicação, permitindo que você se concentre no desenvolvimento de aplicativos.

o Gerenciador de banco de dados MySQL gerencia todos seus dados armazenados alojados . Normalmente é utilizado o terminal ou cmd (windows) para acessar o banco de dados mas a que prefira utilizar softwares com interface grafica .
